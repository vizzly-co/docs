# Deploy Vizzly on Google Cloud Platform (GCP)

To run the Vizzly query engine on GCP, we recommend using [Google's Cloud Run product](https://cloud.google.com/run). You'll need to set the number of minimum instances to at least `1`.

### GCP Query Engine URI
```
eu.gcr.io/vizzly/query-engine
```
