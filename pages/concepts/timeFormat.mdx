# Vizzly managed time formats

The supported date time formats are currently;

- `DD/MM/YYYY`
- `MM/DD/YYYY`
- `DD-MM-YYYY`
- `MM-DD-YYYY`
- `DD MMMM YYYY, hh:mm`
- `dddd DD MMMM YYYY, hh:mm`
- `hh:mm`
- `hh:mm:ss`
- `DD MMMM YYYY`
- `MMMM YYYY`
- `YYYY`
- `YYYY-MM-DD[T]HH:mm:ss`
- `quarter`
- `week`
