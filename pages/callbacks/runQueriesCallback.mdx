# Run queries

Specific to the 'custom' implementation only, this async function is called with a list of Vizzly queries, and expects a list of results formatted in the Vizzly schema.

If you are considering this approach, please get in-touch with us to help guide you through this process.

The example project for a custom implementation [can be found here](https://github.com/vizzly-co/library-examples/blob/e6a7ee3bd167cadec5bbab65fa6d7639d5025c7a/examples/next-js/pages/custom.jsx#L10-L49).
