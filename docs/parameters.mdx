# Parameters

To find parameters such as your organisation or dashboard ID, visit the [studio page](https://app.vizzly.co/studio) and open the embed modal by clicking the code snippet icon in the top right corner of the page.

This will open the embed modal which is shown in the screenshot below. To view the parameters, expand the "Parameters" accordion.

<img width="500px" src="/img/parameters-screenshot.png" />

<br /><br />

> Note, the dashboard ID will change depending on the dashboard you have currently selected from the left side of the page.
